## 技術選択理由

### TanStack Query
- サーバー状態の効率的な管理
- 自動キャッシングとバックグラウンド更新
- オプティミスティック更新対応
- エラーハンドリングの簡素化

### Zustand
- 軽量でシンプルなAPI
- TypeScript完全対応
- Redux DevTools対応
- ボイラープレート最小化

### React Hook Form + Zod
- 高性能なフォーム管理
- 型安全なバリデーション
- 最小再レンダリング
- 統合されたエラーハンドリング

### Vite
- 高速な開発体験
- ES modules対応
- 軽量な本番ビルド
- プラグインエコシステム

## パッケージ.json設定
```json
{
  "type": "module",
  "scripts": {
    "dev": "vite",
    "build": "tsc -b && vite build",
    "lint": "eslint .",
    "preview": "vite preview"
  }
}
```

## 重要な設定ポイント
- **type: "module"**: ES modulesを使用
- **最新バージョン**: 互換性を保ちながら最新機能を活用
- **Bun最適化**: Node.jsではなくBun環境に最適化
